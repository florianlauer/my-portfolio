# Task Completion Checklist

After completing a coding task, run through these steps:

1. **Lint**: `npm run lint` — fix any ESLint errors
2. **Build check**: `npm run build` — ensure no TypeScript or build errors
3. **Visual check**: `npm run dev` and verify in browser if UI was changed
4. **Do NOT auto-commit** — leave changes staged/unstaged for the user to review and commit manually
5. **Report**: Summarize files modified with paths and line numbers
